A Pen created at CodePen.io. You can find this one at http://codepen.io/tdbunting91/pen/NNXPbm.

 This is a very simple quote generator that simply makes an AJAX call to the Mashape famous quotes API to present the user with a randomly generated quote. There is also a button to push the quote out to twitter as a tweet. Built using the Alpha for Bootstrap 4, Google fonts and Font Awesome.
